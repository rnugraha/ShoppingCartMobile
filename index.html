Welcome

This sample project is to have experience on jQuery Mobile 1.3.0 beta 1 version in combination with BackboneJS. Taking shopping cart as the main case, I reused most of the logic from my previous sample project http://rnugraha.github.com/ShoppingCartDemo.

The idea is to have basic shopping cart features in gadget version whereas user can do following:

  • browsing items from some categories
  • add item to cart
  • view shopping cart
  • reset shopping cart
  • remove item from shopping cart

Live demo can be found in : http://www.spicevanjava.nl/shoppingCartMobile/
